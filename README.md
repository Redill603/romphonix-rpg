# romphonix-rpg
A phone collecting RPG!
***Notice: This file is intended for developer use only.***
When working with this project, it's important to know which columns are used for what in the DS Grids.
0 - Name
1 - Level
2/3 - HP/Max HP
4 - Energy
5/6 - EXP/Max EXP
7 - Brand
8 - AT
9 - DF
10 - WT
11 - Icon
12 - Screen status
13 - Board status
14 - Cover status
15 - Battery status
16-19 - Moves 1-4
20 - Sprite
21 - Description
22 - Base EXP
23 - (currently unused)
